## 補足事項

### レスポンシブ対応

- モバイル表示時はテーブルをカード形式に変更
- ナビゲーションはハンバーガーメニューで表示

### バリデーション

- 必須項目（\*）は入力必須
- メールアドレスは形式チェック
- 電話番号は数値・ハイフンのみ許可

### 通知機能（オプション）

- 上長からコメントがあった場合、メール通知
- 日報未提出時、リマインダー通知
